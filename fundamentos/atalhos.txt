(Alt + Shift + Seta para para cima) = suplica a linha

(Ctrl + x) EXCLUIR UMA LINHA
(Ctrl + Alt + Seta para cima / Seta para baixo)ADICIONAR CURSOR ACIMA/ABAIXO
(F2) RENOMEAR SÍMBOLO/ VARIÁVEL ETC...
(Ctrl + P + NOME DO ARQUIVO) PARA PROCURAR UM ARQUIVO
(Alt + seta para cima ou para baixo)Nevegar linhas de códigos para cima e para baixo
